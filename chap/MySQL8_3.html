3.2 - 刪除資料庫

drop database test_db;

3.3 - MySQL 8.0 的新特性 — 系統表全部為 InnoDB表

MySQL 8.0 開始，系統表全部換成交易型的 InnoDB表，預設的 MySQL 實例將不包含任何 MyISAM表，除非手動建立 MyISAM表

select distinct(engine) from information_schema.tables;
