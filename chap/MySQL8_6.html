06 - MySQL 函數

6.1 - MySQL 函數簡介

MySQL 提供大量且豐富的函數，在進行資料庫管理、資料查詢和操作時將經常使用到各種函數。

6.2 - 數學函數

絕對值函數 abs(x)

求2、-3.3和-33的絕對值
select abs(2), abs(-3.3), abs(-33);


傳回圓周率的函數 pi()

傳回圓周率值
select pi();
預設顯示小數數位為6位


平方根函數 sqrt(x)

求9、40和-49的二次平方根
select sqrt(9), sqrt(40), sqrt(-49);
負數沒有平方根


求餘函數 mod(x,y)

求 31除8 、 234除10 和 45.5除6 的餘數
select mod(31,8), mod(234,10), mod(45.5,6);


取得整數的函數 ceil(x) 、 ceiling(x) 和 floor(x)

求 -3.35 和 3.35 的最小整數
select ceil(-3.35), ceiling(3.35);
ceil(x) 和 ceiling(x) 的意義相同，傳回不小於 x 的最小整數值，傳回值轉化為 bigint

求 -3.35 和 3.35 的最大整數
select floor(-3.35), floor(3.35);
floor(x) 傳回不大於 x 的最大整數值，傳回值轉化為 bigint


取得亂數的函數 rand() 和 rand(x)

使用 rand() 產生亂數
select rand(), rand(), rand();

使用 rand(x) 產生亂數
select rand(10), rand(10), rand(11);

提示
rand() 每次產生的亂數是不同的
rand(x) 的 x 為種子值，用來產生重複序列
傳回值範圍皆介於 0 到 1 之間
